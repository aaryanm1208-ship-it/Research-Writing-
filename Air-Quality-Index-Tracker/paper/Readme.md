# Research Paper – Real-Time Air Quality Index Tracking System

This folder contains the IEEE-format research paper written in LaTeX.

## Files
- `main.tex` – Main research paper source
- `architecture.png` – System architecture diagram
- `references.bib` – Bibliography file (optional)

## How to Compile
1. Open Overleaf or any LaTeX editor
2. Upload all files in this folder
3. Compile using PDFLaTeX
